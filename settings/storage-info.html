Storage Info ​

Main Menu → Settings → Storage Info shows how built-in storage (and microSD, when present) is used.

AreaMeaning
InternalUsed / available space on built-in storage (KB)
microSDCard status and usage when a card is inserted

Format Storage ​

To erase all stored projects and reformat storage:

  1. Open Storage Info.
  2. Hold Select (long press) Press Back (short) Hold Select + Back on a volume row to open the side menu.
  3. Choose Format Storage and confirm.

WARNING

Formatting erases all projects on that volume. There is no undo. Export or back up anything you need before confirming.

Factory reset of MIDI settings is separate from format — see Settings overview for clock and global channel options.
