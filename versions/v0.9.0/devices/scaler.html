Scaler

Quantize notes to a scale

When to use it ​

Use Scaler anywhere you need in-key notes — after a random source, before Harmony/Arp, or on live input. Choose a preset scale or build a custom one from individual degrees.

Parameters ​

ParameterRangeWhat it does
Bypass0 – 1When on, the device is disabled and MIDI passes through unchanged.
Key0 – 11Root pitch class (C–B) for scale quantization.
Quantize0 – 2How out-of-scale notes are corrected (down, up, or filtered out).
Scale0 – 50Preset scale from the built-in library, or Custom to use degree toggles.
I0 – 11Include scale degree 1 when Scale is Custom (pitch class 0–11).
I#0 – 11Include scale degree 2 when Scale is Custom (pitch class 0–11).
II0 – 11Include scale degree 3 when Scale is Custom (pitch class 0–11).
II#0 – 11Include scale degree 4 when Scale is Custom (pitch class 0–11).
III0 – 11Include scale degree 5 when Scale is Custom (pitch class 0–11).
IV0 – 11Include scale degree 6 when Scale is Custom (pitch class 0–11).
IV#0 – 11Include scale degree 7 when Scale is Custom (pitch class 0–11).
V0 – 11Include scale degree 8 when Scale is Custom (pitch class 0–11).
V#0 – 11Include scale degree 9 when Scale is Custom (pitch class 0–11).
VI0 – 11Include scale degree 10 when Scale is Custom (pitch class 0–11).
VI#0 – 11Include scale degree 11 when Scale is Custom (pitch class 0–11).
VII0 – 11Include scale degree 12 when Scale is Custom (pitch class 0–11).

ValueNameMeaning

ValueNameMeaning

Quantize ​

ValueNameMeaning
0DownSnap to the nearest scale degree below the input pitch.
1UpSnap to the nearest scale degree above the input pitch.
2FilterDrop notes that are not in the scale.
