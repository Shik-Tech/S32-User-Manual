S32 LFO Parameter ​

Firmware menu name: S32 LFO

Modulate a global S32 LFO parameter.

When to use it ​

Use S32 LFO to modulate one of the 16 global S32 LFOs (rate, depth, sync, wavetable, phase, offset) from a knob — ideal for performance control of modulation depth or speed.

Parameters ​

ParameterRangeWhat it does
Mode—Macro output type. Change this to switch between MIDI, internal routing, and CV16 targets.
LFO IndexLFO 1 – 16Which CV16 or S32 LFO is modulated (depends on macro mode).
CV Param.LFO parameterWhich LFO field the knob adjusts. Available options depend on S32 LFO vs CV16 LFO mode.
MinimumMode-specificValue sent when the knob is at minimum travel. Units follow the target (MIDI 0–127, 14-bit 0–16383, device parameter native range, CV volts, LFO %, etc.).
MaximumMode-specificValue sent when the knob is at maximum travel. Must be greater than or equal to Minimum.
InvertTrue / FalseWhen True, knob clockwise motion maps to decreasing values (minimum ↔ maximum swapped).

S32 LFO parameters you can target ​

When Parameter (CV Param.) is set, these are the LFO fields available in S32 LFO macro mode:

ParameterWhat it does
EnabledTurn the target LFO on or off.
WavetableLFO waveform shape.
RateLFO speed (sync divisions or Hz depending on LFO sync setting).
SyncLock LFO rate to MIDI clock.
DepthModulation depth (0–100%).
OffsetDC offset added to the LFO (-100 – +100%).
PhaseStart position in the cycle (0–360°).
