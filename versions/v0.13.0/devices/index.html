MIDI Device Reference ​

Each device is a block in a rack chain. MIDI flows through devices in order. Select a device in the rack grid to open its parameter list and visualization.

Categories ​

Devices are grouped the same way as on the S32 Add Device picker — alphabetical within each category.

I/O & Clock

  • Clock — BPM, transport, and clock output
  • Midi IN — Receive MIDI from a port
  • Midi OUT — Send MIDI to a port

Generative

  • Arp — Arpeggiate held notes
  • Euclid — Euclidean rhythm generator
  • Seed — Deterministic note sequencer
  • Step Seq — Classic clock-synced step sequencer

Pitch & Harmony

  • Harmony — Chord-aware note transform
  • Scaler — Quantize notes to a scale
  • Transpose — Shift pitch by semitones

Filters

  • CC Type Filter — Pass or block CC types
  • CC Value Filter — Filter CC by value
  • Channel Filter — Filter by MIDI channel
  • Filter — General note/CC filter
  • Note Range Filter — Filter by note range

Routing

  • Dispatch — Fan-out to multiple paths
  • Router — Route by channel or type

Dynamics

  • Envelope — ADSR envelope on MIDI
  • Limiter — Cap velocity or CC peaks
  • Vel. Amp. — Scale note velocity

Effects

  • Delay — Delay notes in time

Utility

  • Chance — Probabilistic note gate
  • Looper — Loop and overdub MIDI
  • M2CV — MIDI to control voltage
  • Random — Randomize note parameters

Adding a device ​

Main Menu → Midi Racks → rack → + cell → pick type from the categorized grid.
