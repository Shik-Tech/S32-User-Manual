External MIDI Mapping ​

Control S32 device parameters and LFO parameters from external MIDI (keyboard, DAW, another controller).

INFO

Overview and how this fits with knob macros: Mapping.

How to map ​

  1. Open the parameter list (device or LFO editor).
  2. Highlight the parameter.
  3. Hold Back and Select together Hold Select + Back → Options → External Mapping.
  4. Display shows: Send a note, CC, or 14-bit CC.
  5. Move the control on your external gear.
  6. Mapping is saved in the active project.

What can be learned ​

MessageBehavior
CCValue 0–127 scales to parameter min–max
14-bit CCMSB + LSB pair auto-detected for smoother control
NoteVelocity scales to range; on/off params toggle on each note-on

INFO

Channel must match the learned message; any input port works.

Badges ​

Mapped parameters show a small badge on the list (CC number or note name):

Knob-macro and LFO badges use different glyphs — see Mapping and Modulation.

Remove mapping ​

Hold Back and Select together Hold Select + Back on the parameter row → Options → Remove Mapping.

The parameter is free for macro or LFO assignment again.

vs knob macros ​

Knob macroExternal mapping
SourceS32 knobIncoming MIDI
SetupPerform → macro gridParameter list → External Mapping
Per parameterOne macro pathOne external source

WARNING

You cannot combine macro and external mapping on the same parameter. LFO modulation is separate — see Modulation.
