Quick Start ​

Power on → Perform screen. You are ready to turn knobs and send MIDI.

Open the Main Menu ​

From Perform: Hold Back and Select together hold Select and Back together → Main Menu.

  • Turn the encoder Encoder — move highlight
  • Press Select (short) Select — open item
  • Press Back (short) Back — return to Perform

Build a rack chain ​

  1. Main Menu → Racks.
  2. Pick a rack (or create one).
  3. Press Select (short) Select a device icon to edit its parameters.

Device chains hold up to 10 devices — see Rack chains.

Configure a knob macro ​

You are on Perform (the default screen after power-on):

  1. Turn the encoder Turn the encoder to choose a knob (1–32).
  2. Press Select (short) Select — enter edit mode.
  3. Turn the encoder Highlight a macro row (1–4), then Press Select (short) Select again to open its parameter grid.

INFO

See Setting up macros for the full walkthrough.
